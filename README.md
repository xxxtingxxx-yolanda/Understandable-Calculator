# 分享网页（share_story_site）

## 位置
`c:\Users\XXXTINGXXX\AndroidStudioProjects\share_story_site`

## 打开方式
- 直接双击 `index.html`
- 或在终端里运行（推荐）：
  - `cd c:\Users\XXXTINGXXX\AndroidStudioProjects\share_story_site`
  - `python -m http.server 8080`
  - 浏览器打开 `http://localhost:8080`

## 文件结构
- `index.html`：页面结构与故事内容
- `styles.css`：视觉样式与响应式布局
- `script.js`：滚动显隐、阶段高亮、进度条
- `assets/`：阶段图片素材（可直接替换同名文件）
